# 使用方法
1. 创建一个 excel 文件，包含两列：name, score；
2. 点击“加载Excel”文件，导入你的成绩单；
3. 拖动参数，**拯救班级的及格率**！
4. 点击“导出修改后成绩”，保存最终的结果吧！
